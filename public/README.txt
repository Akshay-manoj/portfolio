Put your static assets here.

- avatar.jpg        -> your profile photo (square works best). Then set
                       `hasAvatar: true` in lib/data.ts.
- resume.pdf        -> your CV. Then set `resume: "/resume.pdf"` in lib/data.ts.
- projects/*.png    -> project screenshots. Reference them in lib/data.ts as
                       image: "/projects/your-file.png"
